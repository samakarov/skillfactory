Финальный проект: "Tweet sentiment extraction"

Группа: DST-34

Выполнил: Макаров Сергей.

Цель работы: создать и обучить модель позволяющую выделить ключевые фразы, определяющие эмоциональную окраску поста.

Парсинг данных не требуется. Метрика: Jaccard - размер пересечения полученного множества текстовых фрагментов с множеством исходных текстов.

Работа выполнялась на основе предложенного датасета. Запрещено использование подключения к Интернет. Возможно использование датасетов, загруженных на Kaggle.

Заключение:

Зафиксированы навыки работы с RNN и предобученными моделями.
Проведен тщательный feature engineering и отобраны значимые признаки.
Применены предобученные вектора GLOVE.
Применен механизм Transformer. Использовался один из вариантов сети BERT для работы с текстом.
Выполнено ансамблирование одинаковых моделей и получен сабмит с лучшим результатом на Kaggle: 0.71141 / 0.70859

Стоит отметить, что не выполнено несколько пунктов, но точно хотелось бы выполнить:
Применить механизмы агументации исходных данных для увеличения объема данных для обучения моделей. 
Применить ансамбль из различных моделей.
